# DEVCONF-2026

A simple conference landing page built using HTML and CSS.

## About the Project

This is a beginner-level web development project. The website is designed for a fictional developer conference called **DevConf 2026**.

## Features

- Responsive navigation bar
- Hero section
- Meet the Speakers section
- Pricing section
- Developer Challenge section
- Simple and clean design

## Technologies Used

- HTML5
- CSS3
- Visual Studio Code

## Project Structure

```
DEVCONF-2026/
│
├── index.html
├── style.css
├── README.md
└── images/
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
